## ğŸ”„ ANIMATION SYSTEM (ANIMATSIYA TIZIMI)

### Timing Functions
```css
/* Default easing */
transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);

/* Smooth ease-out */
ease-out: cubic-bezier(0, 0, 0.2, 1);

/* Bounce effect */
ease-in-out: cubic-bezier(0.4, 0, 0.6, 1);
```

### Duration Scale
```css
duration-150: 150ms  /* Quick interactions */
duration-300: 300ms  /* Standard transitions */
duration-500: 500ms  /* Smooth animations */
duration-700: 700ms  /* Slow, emphasized */
```

### Animations
```css
/* Spin (Loading) */
@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
animation: spin 1s linear infinite;

/* Pulse (Glow) */
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}
animation: pulse 2s cubic-bezier(0.4, 0, 0.6, 1) infinite;

/* Zoom-in (Icons) */
@keyframes zoom-in {
  from {
    opacity: 0;
    transform: scale(0);
  }
  to {
    opacity: 1;
    transform: scale(1);
  }
}
animation: zoom-in 300ms cubic-bezier(0, 0, 0.2, 1);
```

### Hover Transitions
```css
/* Card hover */
.card {
  transition: all 300ms ease-out;
}
.card:hover {
  transform: translateY(-2px);
  border-color: rgba(71, 85, 105, 0.5);
  box-shadow: 0 25px 50px -12px rgba(15, 23, 42, 0.5);
}

/* Badge hover */
.badge {
  transition: box-shadow 300ms ease-out;
}
.badge:hover {
  box-shadow: 0 10px 25px rgba(34, 197, 94, 0.5);
}

/* Accent line appearance */
.accent-line {
  opacity: 0;
  transition: opacity 300ms ease-out;
}
.card:hover .accent-line {
  opacity: 1;
}
```
